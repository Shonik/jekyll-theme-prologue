--- title: 3dRudder Unity Plugin icon: fa-check order: 6 ---

We designed this Unity plugin to allow developers to integrate our device, the 3dRudder, easily in their games.
It has a custom editor window with dynamic curves made using OpenGL, the possibility to save and load parameters using scriptable objects and a custom live viewer window to see the connected 3dRudder state.
It is available for download on the company developer website.
